# Quantum View

A quiet, single-page site for **Quantum View**, the UK-based Quantum Therapy practice of Charu Sharma. Built with Next.js 14, Tailwind CSS, and a Three.js particle sphere.

## Pages

- `/` — main page: hero, introduction, link to *About the Therapy*, credentials, About Charu, contact form.
- `/about-the-therapy` — secondary page with the longer summary by Ichha B. Talwar and UK therapist contacts.

## Run locally

```bash
npm install
npm run dev
```

Then open <http://localhost:3000>.
